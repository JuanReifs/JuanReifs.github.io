Tecnología con propósito y estrategia

Sin embargo, mi curiosidad no se detiene en el plan de estudios. Por ello, me he especializado en el uso de **Inteligencia Artificial** para la creación de bucles de trabajo y automatización de tareas (formándome con referentes como Jon Hernández). Esta visión, sumada a mi interés por el **SEO**, me permite entender el desarrollo de software no solo como escritura de código, sino como la creación de productos que deben ser visibles, eficientes y escalables.

Aspectos de mi Personalidad

Me considero una persona analítica, metódica y con gran capacidad de aprendizaje autodidacta. Disfruto enfrentándome a nuevos desafíos técnicos y encuentro satisfacción en resolver problemas complejos mediante soluciones elegantes. Valoro el trabajo en equipo y la comunicación efectiva, entendiendo que los mejores proyectos nacen de la colaboración y el intercambio de ideas.

Temario del Curso

Contenidos estudiados durante el primer y segundo trimestre

Sistemas Informáticos

Fundamentos de arquitectura de computadores, sistemas operativos y administración de sistemas

01

Arquitectura de Computadores

Estudio de los componentes fundamentales de un ordenador: CPU, memoria RAM, almacenamiento, buses de datos. Análisis del modelo de Von Neumann y arquitecturas modernas. Comprensión del ciclo de instrucción y jerarquía de memoria.

02

Sistemas Operativos

Introducción a los sistemas operativos: Windows, Linux y macOS. Gestión de procesos, memoria y archivos. Sistemas de archivos (NTFS, ext4, FAT32). Comandos básicos de terminal y shell scripting.

03

Virtualización

Conceptos de virtualización y máquinas virtuales. Uso de VirtualBox y VMware. Creación y configuración de entornos virtualizados. Ventajas de la virtualización: aislamiento, snapshots y portabilidad.

04

Redes y Conectividad

Fundamentos de redes: topologías, protocolos TCP/IP, modelo OSI. Configuración de red en diferentes sistemas operativos. Direccionamiento IP, máscaras de subred y gateway. Herramientas de diagnóstico: ping, traceroute, netstat.

05

Seguridad Básica

Conceptos de seguridad informática: autenticación, autorización, cifrado. Gestión de permisos en sistemas Unix/Linux. Buenas prácticas de seguridad: contraseñas, actualizaciones, copias de seguridad. Introducción a firewalls y antivirus.

06

Administración de Sistemas

Instalación y configuración de sistemas operativos. Gestión de usuarios y grupos. Monitorización del sistema: uso de CPU, memoria y disco. Automatización de tareas mediante scripts. Logs del sistema y resolución de problemas.

Programación

Fundamentos de programación y desarrollo de algoritmos

01

Introducción a la Programación

Conceptos básicos: algoritmos, variables, tipos de datos. Estructuras de control: condicionales (if, else, switch) y bucles (for, while). Operadores lógicos y aritméticos. Diagramas de flujo y pseudocódigo.

02

Funciones y Modularidad

Definición y llamada de funciones. Parámetros y valores de retorno. Ámbito de variables (scope). Recursividad. Ventajas de la programación modular: reutilización, mantenibilidad y legibilidad del código.

03

Estructuras de Datos

Arrays y listas. Matrices multidimensionales. Pilas y colas. Diccionarios/mapas. Operaciones básicas: inserción, eliminación, búsqueda. Complejidad algorítmica y eficiencia.

04

Programación Orientada a Objetos

Clases y objetos. Atributos y métodos. Encapsulación, herencia y polimorfismo. Constructores y destructores. Relaciones entre clases: composición y agregación. Principios SOLID básicos.

1. Resumen Ejecutivo

Título del Proyecto: Portfolio Profesional - Presentación Digital para Candidaturas

Autor: JUan Reifs García

Centro Educativo: CES Juan Pablo Segundo

Fecha: Febrero 2026

Este proyecto consiste en el desarrollo de un sitio web portfolio profesional utilizando exclusivamente HTML5 y CSS3. El objetivo principal es crear una plataforma digital que sirva como carta de presentación profesional, integrando CV digital, carta de presentación, muestras de trabajos realizados y documentación técnica del proyecto.

2. Objetivos del Proyecto

  • Crear un portfolio web profesional y funcional usando HTML5 y CSS3
  • Demostrar competencias técnicas en desarrollo web frontend
  • Presentar de forma efectiva información profesional y académica
  • Implementar diseño responsive adaptable a diferentes dispositivos
  • Aplicar principios de diseño UI/UX para mejorar la experiencia del usuario
  • Documentar el proceso de desarrollo de forma exhaustiva

3. Metodología de Trabajo

El desarrollo del proyecto se ha estructurado en las siguientes fases:

Fase 1

Planificación y Diseño

Definición de estructura del sitio, wireframes, paleta de colores y tipografías. Análisis de requisitos y especificaciones del proyecto.

Fase 2

Desarrollo HTML

Creación de la estructura semántica del sitio. Implementación de todas las secciones requeridas con etiquetas HTML5 apropiadas.

Fase 3

Estilado CSS

Aplicación de estilos, diseño responsive, animaciones y efectos visuales. Optimización para diferentes tamaños de pantalla.

Fase 4

Pruebas y Refinamiento

Testing en múltiples navegadores y dispositivos. Corrección de bugs, optimización de rendimiento y ajustes finales.

4. Tecnologías Utilizadas

HTML5

Lenguaje de marcado para la estructura del contenido. Uso de etiquetas semánticas (header, nav, section, article, footer).

CSS3

Hojas de estilo para diseño visual. Flexbox y Grid para layouts. Animaciones y transiciones CSS. Variables CSS para mantenibilidad.

Google Fonts

Tipografías web profesionales: Playfair Display para títulos y Source Sans 3 para el cuerpo del texto.

Visual Studio Code

Editor de código utilizado para el desarrollo del proyecto.

5. Estructura del Proyecto

El sitio web se compone de las siguientes secciones principales:

  • Página de Inicio (Hero): Presentación impactante con llamadas a la acción
  • CV Digital: Currículum completo con información personal, formación, experiencia y habilidades
  • Carta de Presentación: Documento formal detallando motivaciones y competencias
  • Portfolio de Prácticas: Muestra de proyectos realizados durante el curso
  • Temario: Contenidos estudiados en las asignaturas del curso
  • Memoria: Documentación completa del proyecto

6. Conclusiones

El desarrollo de este proyecto ha permitido aplicar y consolidar conocimientos fundamentales de desarrollo web frontend. Se ha conseguido crear un portfolio profesional completamente funcional utilizando únicamente HTML5 y CSS3, demostrando que es posible crear interfaces modernas y atractivas sin necesidad de frameworks o librerías adicionales.

Entre los aprendizajes más relevantes destacan: el uso efectivo de Flexbox y CSS Grid para layouts complejos, la implementación de diseño responsive mediante media queries, la aplicación de animaciones CSS para mejorar la experiencia de usuario, y la importancia de la semántica HTML para la accesibilidad y el SEO.

Como líneas de trabajo futuro, se plantea la incorporación de JavaScript para añadir interactividad avanzada, la optimización del rendimiento mediante técnicas de lazy loading, y la implementación de un sistema de temas claro/oscuro.
